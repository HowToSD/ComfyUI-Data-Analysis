# SNS Pairplot
Generates a pairplot from a Pandas DataFrame using Seaborn.

Specify a column name for the hue parameter to color the data points based on that column.

## Input
| Name | Data type |
|---|---|
| dataframe | DataFrame |
| hue_column_name | String |
| title | String |
| style | String |
| palette | String |
