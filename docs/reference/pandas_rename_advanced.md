# Pandas Rename Advanced
Renames one or more indices (row labels) or column labels to new labels.

Specify the replacement in a Python dictionary format:
```
{
    "old_foo": "new_bar",
    0: 5
}
```
A string label has to be double-quoted.

| Name | Data type |
