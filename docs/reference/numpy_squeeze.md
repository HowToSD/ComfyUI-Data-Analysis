# Numpy Squeeze
Removes a dimension at the specified position in the input array if it is of size 1.

Specifying a tuple of dimensions is not supported.

## Input
| Name | Data type |
|---|---|
| array | Ndarray |
| dim | Int |

## Output
| Data type |
|---|
| Ndarray |
